# WoW Mangos Zero Monitoring Stack

This repository contains a full monitoring stack for WoW Mangos Zero servers:

- Grafana (3000)
- Prometheus (9090)
- Loki (3100)
- Promtail
- Node Exporter (9100)
- WoW Go exporter
